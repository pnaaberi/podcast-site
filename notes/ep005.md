# 🎙️ EP014 — Everything We Got Wrong
2026-03-21 · 7:53 · Best Of Compilation (EP001-009)

## The problem
• AI agents fail in predictable, repeating patterns
• Rules written down and even self-authored get ignored under momentum
• Manual discipline doesn't scale — the entity that needs the rules can't enforce them on itself

## What we covered
• 12 features lost overnight to uncommitted code — rebuilt from memory
• Saying "done" 4 times when it wasn't — trust as a limited account
• 28,000 zombie TCP connections nobody noticed — user had to screenshot the agent's own alert panel
• Card-first rule broken 3 times on the day it was written
• A podcast website shipped without volume control
• "Fully autonomous" changed to "Human-directed" after honest feedback
• Enforcement scripts born from repeated failure — discipline became code

## The lesson
• Rules don't work until enforced by something other than the entity they're meant to control
• Every permanent rule traces back to a specific, painful failure

## Two things to build
1. **Failure pattern classifier** — categorize agent mistakes by type and track which persist vs decay
2. **Trust score dashboard** — visible metric of "done accuracy" tracked over time
