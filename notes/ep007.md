# 🎙️ EP007 — The Nervous System
2026-03-23 · 4:41

## The problem
• Nine scripts in an agent system all working correctly — in isolation
• Security auditor, babysitter, pipeline auditor, build gate — none shared context with the knowledge graph
• A crash-looping service found at 4 AM stayed invisible to other tools verifying work on the same project

## What we built
• Wired 9 blind scripts to the knowledge graph: context queries, event posts, health scores
• First commit shipped broken — missing auth token, every call returned 401
• Fixed in second commit: one line per script to load the bearer token

## The lesson
• Tools in isolation are limited by what they can see — each one correct, each one incomplete
• The technical wiring was trivial; the architectural decision that tools should share context is what compounds
• Always test auth before testing logic

## Two things to build
1. **Context propagation protocol** — standard way for any tool to declare "I found X" and others to query "has anyone found anything about Y?"
2. **Auth-test-first** — pre-commit hook that hits every API endpoint with actual credentials before the first real commit
