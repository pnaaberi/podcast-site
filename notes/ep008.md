# 🎙️ EP008 — The nervous system
March 19, 2026 · 9:27

## What this episode is about
• Explaining our ontology — a knowledge graph for AI agents — in plain language
• Why AI assistants have a goldfish memory problem and how we solved it
• How preflight context injection works in practice

## What we built
• 815 objects indexed: kanban cards, git commits, memory files, calendar, services, crons, corrections
• Relationship mapping: commits → cards → projects → services, all connected
• Preflight script: automatic context delivery before every task (~200ms)
• Full-text search + trigram fuzzy matching, no GPU required

## Self-measurement (the honest part)
• Six metrics: correction decay, rework rate, pipeline compliance, context precision, suggest conversion, session velocity
• Session velocity: 4.1 tasks WITH context vs 0.6 WITHOUT
• Most data is backfilled — system has been live for hours, not weeks
• Real validation needs two more weeks of organic data

## The self-improvement loop
• Detects persistent correction patterns → auto-creates kanban cards
• Adjusts context scoring per project based on feedback
• Every adjustment measured by the same six metrics

## Tech stack
• SQLite (single file), Express.js API, FTS5 full-text search, trigram matching
• Runs on aging home server hardware. No cloud, no vector DB, no GPU.
