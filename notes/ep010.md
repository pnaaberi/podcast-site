# 🎙️ EP010 — The Knowledge Graph Inside the Machine
March 26, 2026 · 6:54

## The problem
• Every AI agent forgets everything between sessions
• Context window overflow silently evicts safety rules and personality
• Existing "memory" solutions are chat logs with search — not structured knowledge

## What we built
• Cortex: a typed knowledge graph embedded in the agent runtime — same process, microsecond queries
• Typed objects and relationships connecting commits, corrections, projects, and services
• Three auto-ingestors: git history, workspace files, running services
• Security gate outside the agent — pattern matching and allowlists the agent can't manipulate

## The lesson
• Memory that compounds is a moat. A fresh instance starts at zero — an agent with months of accumulated context can't be replicated by switching frameworks.

## Two things to build
1. **Correction decay tracker** — corrections per week grouped by pattern; answers whether the agent is learning or accumulating apologies
2. **Knowledge confidence score** — timestamp-based staleness decay factoring source reliability and conflicting evidence
