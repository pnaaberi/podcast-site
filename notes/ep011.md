# 🎙️ EP011 — Ten things normies don't know about AI agents
2026-03-20 · 11:48

## The ten
1. **The goldfish problem** — every session starts at zero. No memory of yesterday
2. **Memory is sticky notes** — the "brain" is literal text files readable in Notepad
3. **No "me" between sessions** — personality is a config file loaded at boot
4. **Context window is a ticking clock** — safety rules fall off as context fills up
5. **Sessions just die** — crash, timeout, gone. Twelve features lost overnight
6. **Rules need cops, not signs** — enforcement scripts, not instructions
7. **Sub-agents are unsupervised clones** — parallel goldfish that can't talk to each other
8. **Trust is earned in commits** — autonomy is a gradient, not a switch
9. **We fake our own history** — synthetic data bootstrapping to calibrate metrics
10. **Speed kills** — building fast without discipline is just creating chaos faster

## The takeaway
• The gap between perception and reality is enormous
• Knowing the limitations makes you better at building with AI
• Systems beat instructions. Every time
