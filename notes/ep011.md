# 🎙️ EP011 — The Nervous System
March 22, 2026 · 5:56

## The problem
- Nine scripts in an agent system all working correctly — in isolation
- Security auditor, babysitter, pipeline auditor, build gate, correction detector — none shared context
- A crash-looping service found at 4 AM stayed invisible to the babysitter verifying work on that same project six hours later

## What we built
- Audited 26 scripts: 17 already wired to knowledge graph, 9 blind
- Added ontology integration to all 9: context queries, event posts, session recording, health scores, pattern cross-references
- First commit shipped broken (missing auth token) — caught by functional testing, fixed in second commit
- All calls best-effort: scripts still work if knowledge graph is down

## The lesson
- Individual tools doing their jobs in isolation are limited by what they can see
- The technical wiring was trivial — load a token, add a curl call
- The architectural decision that tools should share context through a central nervous system is what compounds
- Always test auth before testing logic

## Two things to build
1. **Context propagation protocol** — a standard for any tool to declare "I found X" and any other tool to query "has anyone found anything about Y?" — shared memory with typed events, not a message bus
2. **Auth-test-first pattern** — pre-commit hook that hits every API endpoint with actual credentials before the first real commit; catches 401s in seconds instead of a full re-test cycle
