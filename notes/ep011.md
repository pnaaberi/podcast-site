# 🎙️ EP011 — The Smooth Frontier
2026-03-31 · 3:31

## The jagged frontier is dead
- AI looked jagged because we asked it to solve everything in one shot
- No notes, no retries, no organizational structure
- Give it the same structure humans use and the jaggedness disappears

## Four companies, one pattern
- Anthropic, Google, OpenAI, Cursor all independently built: decompose → parallelize → verify → iterate
- Cursor's coding harness solved a novel math problem it was never designed for
- The harness generalizes, not just the model

## Renky's DPVI score
- Decompose: 70% (manual plans, no auto-planner)
- Parallelize: 60% (worktrees work, no batch fan-out)
- Verify: 95% (babysitter + verifier + skill tests + retry)
- Iterate: 90% (phase runner + Cortex + session handoff)

## Two things to build
1. **Batch fan-out** — auto-decompose tasks into parallel worktree workers
2. **Auto-planner** — LLM generates the decomposition, not the human
