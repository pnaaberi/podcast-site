# EP012 — Red Lines Are Just Suggestions
2026-04-01 · solo

## The prompt security problem
- Every agent framework has red lines in markdown
- Instructions are not enforcement — the model usually follows them, not always
- Prompt injection, edge cases, confused agents can all bypass prompt rules

## What we built
- PreToolUse hook that intercepts tool calls at the code level
- Two tiers: hard_block (always denied) and ask (user can approve)
- 14 hard-block command patterns + 5 sub-agent restrictions
- Sub-agents can't run sudo, systemctl, ufw, iptables, crontab

## The design principle
- Sub-agents have full tool access + zero privilege escalation
- Interactive users get approval prompts for dangerous ops
- 21 tests verify all three levels

## Two things to build
1. **Rate limiting** — cap tool calls per minute to prevent runaway agents
2. **Audit dashboard** — visualize blocked operations over time
