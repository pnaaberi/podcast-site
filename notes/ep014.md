## The testing gap
- 9 workspace files define agent behavior — but no way to test if the agent follows them
- Trusting the prompt is the same problem as prompt-based security

## Three-layer framework
- Layer 1: Static linter — 8 checks per file, 0-100 scoring
- Layer 2: Behavioral tests — 14 scenarios, 3 assertion types (contains, not_contains, behavior)
- Layer 3: LLM evaluator — strict pass/fail on behavioral qualities
