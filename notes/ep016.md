# 🎙️ EP016 — The Role That Doesn't Exist Yet
2026-04-05 · 4:30

## What Jake observed
- Architecture decisions came from design thinking: "what happens when the user gets anxious?" not "what framework to use"
- Workspace files treated as a design system — scored on readability, rewritten when they fail
- Security built from trust design: "who has access, what's the denial experience?"

## The workflow that matters
- Human designs the system, directs AI to build it, evaluates the output
- Every podcast player detail was directed — captions, notes format, media controls
- The meta-skill: knowing what to ask for and whether the result is correct

## The missing hire
- Companies hire AI engineers OR UX designers — almost never both
- Products where AI feels like a collaborator (not a chatbot in a suit) were shaped by both-brain people
- 16 episodes document this workflow in action
